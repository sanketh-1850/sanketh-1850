### [OBDPlus](https://github.com/sanketh-1850/OBDPlus)  
An **AI-powered vehicle diagnostic assistant** that helps users interpret and fix engine issues efficiently.
- Built a **backend service for OBD-II diagnostics using Python and FastAPI**, exposing APIs to process live vehicle data, fault codes
- Integrated LLM-based inference to generate **grounded, human-readable explanations,** combining structured vehicle metadata with sensor context.
- Designed a **modular, [microservice-style architecture](https://github.com/sanketh-1850/OBDPlusCloud)** separating API handling, data processing, and AI inference for reliability and testability orchestrated by **Docker Compose.**
- Implemented **structured validation, timeouts, and error handling** to ensure stable behavior with noisy or incomplete sensor data.

### [DocBot](https://github.com/UIC-CS480-Fall2025/dash)  
An **LLM-powered developer assistant** designed to help programmers navigate complex documentation efficiently.  
- Created an AI-powered developer assistant using **Python, FastAPI, and LLMs with Retrieval-Augmented Generation (RAG)** to answer natural-language questions across documentation for 10 widely used Python libraries. 
- Grounds responses in verified source content to **prevent hallucinations and improve factual accuracy.** 
- Makes it faster for developers to understand libraries without manually searching documentation.  
- Optimized retrieval performance by **refining chunking and embedding queries to reduce latency and improve answer relevance.**
- Combines **PostgreSQL (user data/logs) with vector embeddings** for precise semantic retrieval.

A **Named Entity Recognition (NER)**‚Äìbased application for **masking personal and medical information**, enabling privacy preserving GenAI workflows while maintaining high-quality outputs.
- Implemented a hybrid pipeline using **regex (emails), phone number detection, spaCy NER, and SciSpacy** to anonymize sensitive data while preserving context.
